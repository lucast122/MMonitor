    MMonitor User Guide

    Introduction:

    MMonitor is a comprehensive metagenome monitoring software designed to process, visualize, and analyze nanopore sequencing data. The software consists of two main components:

    1. A desktop application with a graphical user interface (GUI) for data selection, pipeline execution, and database management.
    2. A web-based dashboard (django-dash) for result visualization.

    Getting Started:

    Launching the Application:

    • Navigate to the MMonitor root directory and run the __main__.py file.
    • The main GUI window will appear, offering several functionalities.

    Local Database Management:

    • Create Local DB: Set up a new local database for your project.
    • Choose Local DB: Select an existing local database.
    • Start Offline Monitoring: Begin the offline monitoring process.

    Webserver Authentication:

    • Click on the User Authentication button.
    • Provide your username and password registered on the MMonitor webpage.
    • Ensure the 'host' and 'database' fields remain unchanged.

    Data Addition:

    • Add Metadata from CSV: Append metadata by uploading a CSV file. A sample file, 'meta.csv', is available in the root directory for reference.
    • Process Sequencing Files: Process and analyze your sequencing files. Select the desired analysis type and proceed.

    Web Dashboard:

    • Access the MMonitor web dashboard through your web browser.
    • The dashboard provides various visualization options:
      • Taxonomy: View the taxonomic profile of your samples.
      • Correlations: Analyze correlations between taxonomies and metadata.
      • Other visualizations, including horizon plots, statistics, etc.

    Tooltips:

    Hover over interface elements to view tooltips, which provide additional context and instructions for each function.

    Closing the Application:

    Click on the Quit button to safely close the application.

    Note: Always ensure your data is backed up before starting any analysis. MMonitor is a powerful tool, but as with all software, understanding its functions and regularly saving your work will help ensure the best experience.

    For more details and updates, visit the MMonitor repository on GitHub or check out the latest release.
